# Трекер привычек

Курсовой проект по JavaScript.

## Функционал

- Добавление привычек (название, время)
- Отметка выполнения за любой день (клик по дню в календаре или кнопка «Выполнено сегодня / Отменить»)
- Визуализация прогресса:
  - Календарь с галочками на выполненных днях
  - Статистика: процент выполнения за неделю, месяц, общий процент, серия дней подряд
  - Топ самых успешных привычек
- Управление: редактирование, удаление, сброс всей статистики
- Экспорт данных в CSV
- Тёмная тема (переключатель)
- Сохранение в localStorage

## Установка и запуск

1. Скачайте или клонируйте репозиторий - git clone git@github.com:ALex2BRO/Kursach.git.
2. Откройте `index.html` в браузере (желательно через локальный сервер, например Live Server).

## Использование

- **Добавление привычки**: нажмите на любой день в календаре, введите название и (опционально) время, нажмите «Сохранить».
- **Отметка выполнения**: клик по дню в календаре или кнопка «Выполнено сегодня / Отменить» в списке (текст меняется).
- **Редактирование**: ✎.
- **Удаление**: 🗑.
- **Сброс статистики**: кнопка «Сбросить статистику».
- **Экспорт CSV**: кнопка «Экспорт CSV».
- **Тёмная тема**: 🌙.

## Структура проекта

- `storage.js` — localStorage
- `habitManager.js` — бизнес-логика
- `calendar.js` — рендеринг календаря
- `ui.js` — отображение списка, попапов
- `main.js` — инициализация
